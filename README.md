## General AI Applications

AI-native applications designed around conversational, generative, or agent-based interaction.

- [ChatGPT](https://chat.openai.com/) — Conversational AI application for writing, reasoning, coding, and general tasks.
- [Claude](https://www.anthropic.com/claude) — AI assistant focused on long-form reasoning, analysis, and writing.
- [Perplexity](https://www.perplexity.ai/) — AI-native search and answer engine combining retrieval and generation.
- [Pi](https://pi.ai/) — Personal AI focused on conversational interaction and emotional intelligence.

## AI for Writing & Content

Applications where AI is the primary interface for creating and refining written content.

- [Jasper](https://www.jasper.ai/) — AI content platform for marketing, blogs, and brand-aligned writing.
- [Copy.ai](https://www.copy.ai/) — AI writing tool for marketing copy, emails, and workflows.
- [Writesonic](https://writesonic.com/) — AI platform for content generation and SEO-focused writing.
- [Notion AI](https://www.notion.so/product/ai) — Integrated AI writing and editing within a knowledge workspace.
- [Sudowrite](https://www.sudowrite.com/) — AI writing assistant for creative and narrative content.

## AI for Coding & Development

Applications where AI is central to the software development experience.

- [Cursor](https://www.cursor.sh/) — AI-first code editor with integrated agent capabilities.
- [GitHub Copilot](https://github.com/features/copilot) — AI pair programmer integrated into development environments.
- [Replit Ghostwriter](https://replit.com/site/ghostwriter) — AI coding assistant built into a cloud development platform.
- [Codeium](https://www.codeium.com/) — AI coding assistant for autocomplete and chat-driven development.

## AI for Design & Media

AI-native tools for generating and editing visual, audio, and multimedia content.

- [Midjourney](https://www.midjourney.com/) — AI image generation platform focused on artistic and creative outputs.
- [DALL·E](https://openai.com/dall-e) — AI system for generating images from text prompts.
- [Runway](https://runwayml.com/) — AI platform for video generation, editing, and media production.
- [Canva AI](https://www.canva.com/) — AI-powered design tools integrated into a visual content platform.

## AI for Productivity & Knowledge Work

AI-native tools designed to support thinking, organization, and decision-making.

- [Mem](https://mem.ai/) — AI-powered workspace for capturing, organizing, and retrieving knowledge.
- [Tana](https://tana.inc/) — AI-native knowledge graph and note-taking system.
- [Taskade](https://www.taskade.com/) — AI-driven productivity platform combining tasks, notes, and agents.
- [Coda AI](https://coda.io/product/ai) — AI-powered documents and workflows platform.

## AI Assistants & Agents

Applications focused on autonomous or semi-autonomous task execution.

- [AutoGPT](https://github.com/Significant-Gravitas/AutoGPT) — Experimental autonomous AI agent framework.
- [AgentGPT](https://agentgpt.reworkd.ai/) — Browser-based platform for deploying autonomous agents.
- [BabyAGI](https://github.com/yoheinakajima/babyagi) — Minimal AI task management system using agents.
- [Superagent](https://github.com/homanp/superagent) — Platform for building and deploying AI agents.

Tools for creating AI-native applications and experiences.

- [OpenAI Platform](https://platform.openai.com/) — APIs and tools for building AI-native applications.
- [Vercel AI SDK](https://sdk.vercel.ai/) — Toolkit for building AI-powered user interfaces and applications.
- [LangChain](https://github.com/langchain-ai/langchain) — Framework for building LLM-powered applications.
- [LlamaIndex](https://github.com/jerryjliu/llama_index) — Data framework for connecting LLMs to applications.

## Design Patterns & Concepts

Concepts and emerging patterns shaping AI-native applications.

- Prompt-driven interfaces — Applications where user input is interpreted dynamically via natural language.
- Generative UI — Interfaces that adapt or generate layouts based on user intent or data.
- Agent workflows — Systems where tasks are executed through autonomous or semi-autonomous agents.
- Human-in-the-loop — Hybrid systems combining AI automation with human oversight.
- Retrieval-augmented generation (RAG) — Combining external data sources with generative models.
